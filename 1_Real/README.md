## Purpose
Every project must begin with a clear, measurable goal. This folder contains high-level objectives and key results (OKRs) to keep work aligned with tangible outcomes.

## Contents

- Key performance indicators (KPIs)
- Measurable outcomes and milestones

## How to Use
1. Start here before any development work
2. Review objectives regularly to ensure alignment
3. Update OKRs as the project evolves
4. Use these goals to guide decision-making throughout development
